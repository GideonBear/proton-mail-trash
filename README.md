# proton-mail-trash

## Installation
```bash
# Using uv:
uv tool install proton-mail-trash
# Using pipx:
pipx install proton-mail-trash
# Using pip:
pip install proton-mail-trash
```

## Usage
```bash
proton-mail-trash
```
